## Install
(Only Arch and Arch-based, yay requrie)
```bash
git clone https://github.com/roxysqq/rise-hyprland
cd rise-hyprland
./install.sh
```
## Uninstall
```bash
./rise-hyprland/uninstall.sh
```

## Keybinds
```
Super+W - Zen Browser
Super+Q - Exit app
Super+D - Launch apps
Super+T - Telegram
Super+E - File Manager
Super+Shift+<right/left> - Move windows to waorkspase
Super+<right/left> - Move focus
```
